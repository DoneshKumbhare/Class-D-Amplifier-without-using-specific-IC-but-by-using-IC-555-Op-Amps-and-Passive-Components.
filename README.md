# Class-D-Amplifier-without-using-specific-IC-but-by-using-IC-555-Op-Amps-and-Passive-Components.
Simulation of a Class D audio amplifier designed in Proteus 8 using a 555 Timer, Op-Amp, and passive components. The project demonstrates PWM generation, MOSFET switching, and output waveform observation on a virtual oscilloscope, emphasizing component-level circuit design and signal analysis.

Overview
This project demonstrates the design and simulation of a Class D audio amplifier in Proteus 8. Unlike traditional amplifier designs that rely on specialized ICs, this project uses fundamental components: a 555 Timer, an Op-Amp (LM386/LM393), MOSFETs (IRFZ44N), and passive components. The output is observed on a virtual oscilloscope to study waveform behavior.
The project highlights PWM generation, switching control, and audio signal amplification in a digital amplifier topology, providing hands-on understanding of signal processing at the hardware level.

Circuit Diagram
1. 555 Timer: Generates PWM for switching the MOSFETs.
2. Op-Amp LM386: Amplifies the input audio signal.
3. MOSFETs (IRFZ44N): Act as high-speed switches in the Class D topology.
4. Inductor & Capacitor (L1 & C4): Forms the output filter to reconstruct the analog waveform.
5. Oscilloscope: Used to monitor PWM signals and output waveform.

Simulation Output
1. Channel A: Input audio waveform.
2. Channel B: PWM waveform generated by the 555 timer.
3. Channel C/D: Amplified output waveform after MOSFET switching and LC filtering.

Key Features

2. Demonstrates PWM-based audio amplification.
